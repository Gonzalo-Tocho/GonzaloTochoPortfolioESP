Análisis exploratorio COVID-19 en SQL

Este proyecto se enfoca en el análisis de datos relacionados con el COVID-19, incluyendo casos totales, muertes, tasas de mortalidad y vacunación a nivel global y por país. Se realizan cálculos de tasas de infección y mortalidad, se analiza la tendencia de casos y muertes por día y se explora el impacto de la vacunación en la población.

Análisis de ventas de pizza en SQL

Este proyecto analiza datos de ventas de una pizzería ficticia, utilizando consultas SQL para extraer información relacionada con los ingresos, tendencias de pedidos y popularidad de las pizzas. El conjunto de datos incluye información sobre las ventas de pizzas, como detalles de los pedidos, categorías de pizzas, tamaños y cantidades vendidas.

Tableau Dashboard Vehículos Eléctricos

Este proyecto de Tableau analiza datos sobre vehículos eléctricos en los Estados Unidos. Los análisis incluyen el número total de vehículos por año de modelo, los estados con más vehículos eléctricos, los principales fabricantes, y la tasa de elegibilidad para programas de incentivos. El dashboard también proporciona información interactiva, permitiendo filtrar por tipo de vehículo, modelo y estado.
